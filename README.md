# 算法过程可视化系统

面向数据结构、离散数学、程序设计相关课程学习者的交互式算法可视化实验平台。

## 系统简介

本系统支持用户自主选择算法、自定义或随机生成测试数据，通过动画、图形、表格、步骤列表等多元化形式，动态展示算法完整执行流程与关键状态变化，同时配套算法原理、复杂度解析、测试用例等辅助学习内容。

### 已实现算法（3类3种）

| 算法类型 | 具体算法 | 难度等级 |
|----------|----------|----------|
| 排序算法 | 快速排序 | 低-中 |
| 图算法 | Dijkstra 最短路径 | 中 |
| 递归算法 | 汉诺塔 | 中-高 |

## 系统完整运行部署方式

### 环境要求

- **Node.js** >= 18.x
- **npm** >= 9.x

### 安装与运行

```bash
# 1. 进入项目目录
cd algorithm-visualization-system

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 访问应用
# 浏览器打开 http://localhost:3000
```

### 构建生产版本

```bash
npm run build       # 构建到 dist/ 目录
npm run preview     # 预览生产版本
```

## 算法选择操作流程

1. **启动系统**：执行 `npm run dev`，浏览器自动打开 `http://localhost:3000`
2. **选择算法**：在左侧面板"算法选择"区域，点击目标算法卡片（快速排序 / Dijkstra最短路径 / 汉诺塔）
3. **输入数据**：
   - **手动输入**：在输入框中按格式录入数据
   - **随机生成**：点击"随机生成"标签自动填充合规数据
   - **测试用例**：点击预设测试用例直接加载
4. **执行算法**：点击「▶ 执行算法」按钮
5. **观看演示**：系统自动展示首步状态，可通过播放控制栏操作
6. **查阅信息**：右侧面板实时显示步骤列表，左侧显示算法原理和复杂度信息

## 测试数据输入格式规范

### 快速排序

- **格式**：用逗号（中英文均可）、空格分隔的整数序列
- **示例**：`64, 34, 25, 12, 22, 11, 90`
- **限制**：2-20 个整数元素

### Dijkstra 最短路径

- **格式**：单个大写字母（A-G），表示起始节点
- **示例**：`A`
- **图结构**：系统内置固定图结构，包含 7 个节点和多条加权边

### 汉诺塔

- **格式**：整数（2-8），表示盘子数量
- **示例**：`4`
- **说明**：数字越大，盘子越多，移动步骤呈指数增长

## 项目技术栈

- **前端框架**：React 18
- **构建工具**：Vite 5
- **图形渲染**：SVG（Dijkstra图、汉诺塔柱子）
- **动画**：CSS Transitions + Animations
- **语言**：JavaScript (JSX)
